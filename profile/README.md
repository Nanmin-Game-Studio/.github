# Nanmin Game Studio

**Indie game studio. Currently building Hēlapidi — a Sri Lankan-themed multiplayer arena shooter.**

Based in Sri Lanka. Founded and run by Madushan — solo developer, designer, and engineer.

---

## 🎮 Hēlapidi

A Sri Lankan-themed 2D multiplayer arena shooter. Fast movement, chaotic real-time combat, built around four elemental characters rooted in local cultural identity.

- **Status:** In active development
- **Stack:** JavaScript, Phaser.js, Node.js, Socket.io
- **Repo:** [`Nanmin-Game-Studio/helapidi`](https://github.com/Nanmin-Game-Studio/helapidi)

---

## 🚀 What We Do

- Multiplayer game development (WebSockets / real-time systems)
- Gameplay mechanics, physics, and feel-driven design
- Cultural storytelling

## 🧠 Tech Stack

`JavaScript` · `Phaser.js` · `Node.js` · `Socket.io` · `HTML5` / `CSS3`

## 📌 Mission

To build Hēlapidi into a polished, mechanically sharp multiplayer game that proves a culturally rooted indie title can stand on its own in the global indie scene.
