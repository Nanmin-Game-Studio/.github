# .github

This is the meta repository for the **Nanmin Game Studio** GitHub organization.

It does two jobs:

1. `profile/README.md` — the content rendered on the org's public homepage at
   https://github.com/Nanmin-Game-Studio
2. `.github/workflows/` — automation that generates the visual assets
   (`cube.svg`, `pipeline.svg`) referenced in that homepage.

This repo is **not** a game. It's infrastructure for the org page. The actual
game repo lives separately under this org (`helapidi`).
